Project Ankitan

National Land Knowledge & Dispute Resolution Repository supporting SVAMITVA implementation through verified historical and transactional land data integration.

Created by Kallol Chakrabarti, Global Independent Researcher

Overview

Project Ankitan augments digitized village land records by integrating verified legacy, transactional, and community-based land knowledge. It strengthens title clarity, supports dispute resolution, and enhances land-based financing readiness.

Problem Addressed

  • Misalignment between historical boundaries and digitized maps
  • Unrecorded or informal land transactions
  • Fragmented legacy records
  • Dispute delays impacting property card issuance
  • Limited credit access due to title uncertainty

Core Objectives

  • Improve title certainty under SVAMITVA
  • Reduce rural land disputes
  • Enable credit-linked land assets
  • Provide a scalable integration framework for all states

Implementation Phases

  • Data ingestion and verification
  • Geo-referencing and reconciliation
  • Dispute analytics and flagging
  • Credit-readiness tagging
